# Инструкция по сборке .exe файла

## Быстрый запуск (без .exe)

Самый простой способ - использовать файл `start.bat`:
1. Дважды кликните на `start.bat`
2. Приложение откроется в браузере автоматически

## Сборка .exe файла

### Шаг 1: Установка зависимостей

```bash
npm install
```

Это установит все необходимые пакеты, включая Electron и electron-builder.

### Шаг 2: Сборка приложения

```bash
npm run electron:build
```

Это создаст:
- Папку `release/` с установщиком Windows
- Файл `ChessTrainer Setup X.X.X.exe` - готовый установщик

### Шаг 3: Установка

1. Запустите `ChessTrainer Setup X.X.X.exe`
2. Следуйте инструкциям установщика
3. Приложение появится в меню "Пуск" и на рабочем столе

## Альтернативные команды

### Разработка с Electron (для тестирования)

```bash
npm run electron:dev
```

Запустит Electron приложение в режиме разработки.

### Сборка без установщика (только папка)

```bash
npm run electron:pack
```

Создаст папку с готовым приложением в `release/win-unpacked/`, которую можно запускать напрямую.

## Распространение

После сборки в папке `release/` будет:
- `ChessTrainer Setup X.X.X.exe` - установщик (рекомендуется)
- `win-unpacked/` - папка с готовым приложением (можно копировать целиком)

### Для распространения на других компьютерах:

**Вариант 1 (рекомендуется):**
- Скопируйте файл `ChessTrainer Setup X.X.X.exe`
- На другом компьютере запустите установщик
- Node.js НЕ требуется!

**Вариант 2:**
- Скопируйте всю папку `win-unpacked/`
- Запустите `ChessTrainer.exe` из этой папки
- Node.js НЕ требуется!

## Требования для сборки

- Node.js 16+ (только на компьютере, где собираете)
- Windows 10/11 (для сборки Windows версии)

## Требования для запуска .exe

- Windows 10/11
- Node.js НЕ требуется!

## Размер файлов

- Установщик: ~150-200 МБ
- Распакованная папка: ~200-250 МБ

Это нормально для Electron приложений, так как они включают в себя встроенный браузер Chromium.
